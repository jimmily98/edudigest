# Project Structure

- `src/backend`: Node.js/Python backend for handling uploads and processing.
- `src/frontend`: User interface for uploading materials and interacting with the system.
- `src/utils`: Helper scripts for audio extraction (FFmpeg) and ASR (Whisper/Gemini).
- `data/uploads`: Temporary storage for uploaded materials.
- `data/transcripts`: Storage for the final PDF transcripts.
